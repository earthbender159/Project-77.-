Visual studio code is something that you can use to code and design yuor own websites. To set it up all you have to do is download the platform. Next, you can click on the extentions button on the left side and search "live server". You can then install the download so you will be able to run or test uyour websites. Finally, you can create a new folder in your files and you can click file, then open folder. After you should be able to select a folder. By pressing the new file button on the name of the folder you should be able to create a html, css, or other file.


To open the console screen look at the bottom of your screen where you should see the sections "problems", "output", "terminal", and "debug console". Click on "debug console" and you should be able to see the console. To print things on the console you can create a variable and afetr you can use console.log to and in parentheses the name of your variablethat you would like to show.
